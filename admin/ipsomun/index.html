파이프라인 단계별 실행

개별 단계를 수동으로 실행합니다. 기존 cron 스케줄에 영향을 주지 않습니다. 


        

      


      

        
전체 파이프라인 실행

        

          수집부터 발행까지 전체 파이프라인을 순차 실행합니다. (약 5~10분 소요)
        

        
      


      

        
오늘 피드 재발행

        

          오늘 발행된 피드를 삭제하고, 아이템을 draft로 되돌린 뒤 publish-daily를 다시 실행합니다.
        

        
      


      

        
특정 날짜 재발행

        

          특정 날짜의 피드를 삭제하고 아이템을 draft로 되돌립니다. 이후 수동으로 publish-daily를 실행하세요.
