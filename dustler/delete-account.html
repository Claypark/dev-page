Account & Data Deletion Request

Dustler(더슬러) 앱에서 사용 중인 계정과 관련 데이터의 삭제를 요청할 수 있습니다.

삭제되는 데이터

계정 삭제 요청 시 아래의 모든 데이터가 삭제됩니다.

  • 계정 정보 — 이메일 주소, 표시 이름, 프로필 사진
  • 청소 일정 데이터 — 등록한 집, 공간, 청소 작업, 완료 기록
  • 가족 그룹 정보 — 가족 구성원 연결 및 초대 기록
  • 디바이스 정보 — 푸시 알림용 FCM 토큰

삭제된 데이터는 복구할 수 없습니다. 삭제 요청 전 신중하게 결정해 주세요.

삭제 요청 방법

방법 1. 앱 내에서 삭제

  1. Dustler 앱을 실행합니다.
  2. 설정(Settings) 메뉴로 이동합니다.
  3. "계정 삭제"를 탭합니다.
  4. 안내에 따라 삭제를 확인합니다.

방법 2. 이메일로 요청

아래 이메일로 삭제 요청을 보내주세요. 가입 시 사용한 Google 계정 이메일에서 발송해 주시면 빠르게 처리됩니다.
